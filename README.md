# EV Energy Consumption Prediction 

## Project Overview
This project predicts EV energy consumption using Random Forest Regressor.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Gradio

## Features
- Data preprocessing
- Feature engineering
- Random Forest model training
- Model evaluation
- Interactive prediction web app using Gradio

## How to Run

```bash
pip install -r requirements.txt
python app.py
